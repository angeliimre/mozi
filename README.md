# About the project
With this page people can book a seat in a cinema.<br/>
If you select a place by clicking the place of the seat, it will become yellow if it is grey, otherwise you will get a warning.<br/>
By clicking "Küldés" button, you can finish your resercation.<br/>
The app will store the positions of the booked seats in a txt file
